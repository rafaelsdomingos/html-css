Principais Formatações

Negrito/Destaque

Nesta frase temos um termo em negrito usando a tag <b> (Não semântica)

Nesta frase temos um termo em negrito usando a tag <Strong> (semântica)

Itálico/Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (Não semântica)

Nesta frase temos um termo com ênfase usando a tag <em> (semântica)

Marcação de texto

Podemos também usar um texto marcado usando a tag <mark>

Texto grande e texto pequeno

Estamos criando um texto grande com a tag <big>(já em desuso) e um texto pequeno com a tag <small>

Texto exluído

Podemos marcar um texto como excluído com a tag <del>, para indicar que ele deve ser lido mas ignorado.

Texto incluído(sublinhado)

Podemos marcar um texto como incluído com a tag <ins>, para indicar que o texto foi adicionado recentemente. Tem o mesmo efeito visual que a tag <u>, mas essa é uma tag não semântica.

Texto sobrescrito

Usamos a tag <sup> para deixar um texto sobrescrito x2+2 = 0

Texto subscrito

Usamos a tag <sub> para deixar um texto subscrito H2O
